to extract tree, commit, blobs, or tags form a pack:
echo <ID> | pack-objects
This only works one 1 obj at a time.
However if I push the <ID> into a file, 1 ID per line, I can then cat the results out; and get a pack of those iteams.



pack a commit: 
git rev-list --objects  -1 <Commit id> | awk '{print $1}' | git pack-objects --stdout > o3.pack





to find contents of a pack file:
1) git index-pack <file>.pack
2) git verify-pack -v <file>.pack



to exploded pack into system:
git unpack-objects <pack>
